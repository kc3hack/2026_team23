## 背景・課題・解決されること

集団で旅行をする際こんな悩みありませんか？
**1か月前から言ってるのに誰も解答してくれない..**
それは、旅行主催者によくある悩みかと思います。そこで私たちは解決策を提案します。

この現象の解決法はズバリ...もっと前に参加者に知らせて予定をあらかじめロックすればよいのです！

**空いてる予定に合わせるのではなく3か月前から予定を確定させる。**
そのために便利な機能をたくさん実装しました。

## プロダクト説明

Googleログインをし、まずは利用登録をしましょう。
参加者全員がアカウント登録を終えたら、旅行に行きたいメンツを決めてグループ作成しましょう。
グループ作成が終わったらいよいよ旅行の計画を立てましょう。
日程は3か月先しか入力できません。
=>3か月後なら調整することなくほとんどの方が予定空いてるでしょう！
作成すると招待された参加者にpush通知が届きます。
通知が届いたら参加者は参加を押すだけ。

全員の承認を得られたら旅行の日程確定です！

でも... 数人はめんどくさがって回答してくれない..
=>そんなヤツには追い通知を送ってみましょう。通知内容をカスタマイズできます。
